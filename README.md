# SurvivalHorde
WIP Unity game

A game with the likeness of Vampire survivors and magic survival

Currently Implemented:
- 3 abilities: Shoot, Slash, Dash
- Leveling system
- Ability upgrade
- Simple enemy AI
- Enemy pooling and scaling spawn rate
- Psuedo Endless walk area
